  • loadConfig: (options: LoadOptions, defaults?: unknown, refresh?: boolean) => any
      • (options: LoadOptions, defaults?: unknown, refresh?: boolean): any

      • Load config files from a directory.

        • the directory can contain multiple files, each has a partial config. this function will load the files according to various environment settings and compose them into a single one.

        • Each config file in the directory can be JSON, JS, Yaml, or TS. For JS or TS, if it's ES6 Module and has default export, then it's used as config partial, else the whole module will be used.

        • The result is cached using the full directory path

        Parameters

        • options: LoadOptions

          LoadOptions for loading config files

        • Optional defaults: unknown

          default config values

        • Optional refresh: boolean

          refresh the cached copy - NOTE this will clear any values user set in the config

        Returns any

        returns config
